## Что это за проект

Desktop-приложение для интерактивной оптимизации транскриптов и управления YAML-словарями очистки. Стек: Tauri 2 + React 19 + TypeScript + Vite + Monaco Editor + Zustand + Rust.

- Нет ограничения TTL при расшифровке (соответствует Python без TTL).

---

## Key decisions & constraints

- **Fernet в чистом Rust**: `fernet` crate отказан, т.к. требует системный OpenSSL. Используем `aes` + `cbc` + `hmac` + `sha2` + `base64` + `rand`.
- **Порядок байт ключа**: как в Python cryptography — `key[:16]` = signing key, `key[16:]` = encryption key.
- **Расшифровка в Rust**: фронтенд не видит plaintext API-ключ; Fernet-ключ только в env.
- **Backward compatibility**: старый плоский `settings.llm` → `models.default`.
